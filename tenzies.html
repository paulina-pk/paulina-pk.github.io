TenziesDice Game

Project overview

Digital dice game where players roll until all ten dice match. Built during Scrimba's Learn React course with additional features including custom dot-pattern dice, roll tracking, and local storage for best scores.

Technologies used

  • React
  • JavaScript
  • HTML/CSS
  • Responsive design

Key features

Players click to hold selected dice, freezing them while re-rolling others until all match. Roll counter with local storage tracking lowest score across sessions.

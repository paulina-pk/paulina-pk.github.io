QuizzicalKnowledge Test

Project overview

Interactive quiz app fetching questions from an external API. Developed during Scrimba's Learn React course from a Figma design, allowing users to answer questions, check results, and play again with new content.

Technologies used

  • React & React Hooks
  • JavaScript
  • HTML/CSS
  • Fetch API
  • Responsive design

Key features

Five questions fetched from an external API with multiple-choice or true/false answer options. Answer validation with color-coded feedback: green for correct, red for incorrect selections. Play again functionality generates new questions from API.

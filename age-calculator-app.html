AgeCalculatorApplication

Project overview

A simple age calculator built from a Frontend Mentor design challenge. Users input their birthdate, and the application calculates their age in years, months, and days with animated number displays and comprehensive input validation.

Technologies used

  • JavaScript
  • HTML/CSS
  • Responsive design
  • Mobile-first workflow

Key features

Input validation with dynamic error messages for empty or invalid fields. Calculates age in years, months, and days with animated number displays.

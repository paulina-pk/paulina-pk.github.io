HangmanGame

Project overview

A classic word-guessing game built with React. Built as a challenge for Udemy's Modern React Bootcamp, featuring custom React components, state management with hooks, and responsive design with original styling.

Technologies used

  • React.js & React Hooks
  • JavaScript
  • HTML/CSS
  • Responsive design

Key features

Players guess letters using alphabet buttons. Correct guesses reveal letters; incorrect guesses progressively draw the hangman. After six wrong guesses, the game ends with a loss. Successfully guessing the word displays a win message

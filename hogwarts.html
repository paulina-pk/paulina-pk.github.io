Hogwarts SchoolDemo Website

Project overview

A full-stack demo website for the Harry Potter universe, featuring a digital enrollment system for first-year students. Created as the final project for Harvard's CS50 course, combining frontend interactivity with backend data management using Python, Flask, and SQLite.

Technologies used

  • Python & Flask
  • Jinja templating
  • SQLite3
  • JavaScript
  • AJAX / JSON
  • Bootstrap

Key features

Custom registration process with Hogwarts-themed modals and personalized acceptance letter. Dynamic Sorting Hat quiz with tie-breaker functionality to determine student house placement. Student panel with subject enrollment system and database-driven curriculum display.

Learning outcomes

  • JSON data handling and API integration
  • Frontend-backend communication with Fetch API
  • Database integration (SQLite)
